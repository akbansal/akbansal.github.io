Tip of the Day

Internet Explorer in kiosk mode

Right-clicking on desktop. Select New --> Shortcut. Enter "C:\Program Files\Internet Explorer\IEXPLORE.EXE" -k. Name it IE Kiosk

Of course, you have to substitute keystrokes. e.g. for Back button - use Backspace key. for Forward button - use Alt+Right Arrow.
